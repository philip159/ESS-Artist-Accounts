Product Add-ons Widget

Installation guide for Shopify themes

Quick Start

Add the widget to any product page template to display upgrade options like Box Frame and Paper upgrades.

Recommended: Create a separate product template for testing before applying to live products.

Installation Steps

1

Create a Test Product Template

In Shopify Admin, go to Online Store → Themes → Edit Code

Create a new template: Templates → Add a new template → product → product.addons-test.liquid

2

Add the Container Element

In your product template, add this container below the variant selector:

<!-- ESS Product Add-ons -->
<div id="ess-addons-container" 
     data-product-id="{{ product.id }}">
</div>
3

Add the Script

Add this script tag before the closing </body> tag or in your template:

<script src="https://east-side-studio-london-ar.replit.app/shopify-addons-widget.js"></script>
4

Assign Template to Test Product

Go to a test product in Shopify Admin and change its template to product.addons-test

Complete Liquid Snippet

Copy this entire snippet into your product template:

{%- comment -%}
  ESS Product Add-ons Widget
  Place below variant selector, above Add to Cart button
{%- endcomment -%}

<div id="ess-addons-container" 
     data-product-id="{{ product.id }}"
     style="margin: 20px 0;">
</div>

<script>
  (function() {
    // Load the add-ons widget
    var script = document.createElement('script');
    script.src = 'https://east-side-studio-london-ar.replit.app/shopify-addons-widget.js';
    script.async = true;
    document.body.appendChild(script);
  })();
</script>

Restricting to Specific Products

By default, add-ons with allowedProductIds set will only appear on those products.

Set Product Restrictions via API:

PATCH /api/admin/addons/:id
{
  "allowedProductIds": ["15079516209529", "15079516242297"]
}

Or leave allowedProductIds as null to show on all products.

Styling Customization

The widget inherits CSS variables from your theme. Override these in your theme CSS:

:root {
  --color-foreground: #1a1a1a;
  --color-foreground-secondary: #666;
  --color-background: #fff;
  --color-background-secondary: #fafafa;
  --color-border: #e0e0e0;
  --color-accent: #f0f0f0;
}

Cart Behavior Options

Customize how the widget handles cart additions:

ESSAddons.init({
  productId: '{{ product.id }}',
  
  // Disable redirect to cart page (for drawer/modal carts)
  redirectToCart: false,
  
  // Custom success handler
  onCartSuccess: function(cartData, items) {
    // Trigger your theme's cart drawer
    document.dispatchEvent(new Event('cart:refresh'));
  },
  
  // Custom error handler
  onCartError: function(error, items) {
    console.error('Cart error:', error);
  }
});

Listen for cart events (when redirectToCart: false):

document.addEventListener('ess:cart:added', function(e) {
  console.log('Items added:', e.detail.items);
  // Open cart drawer, update cart count, etc.
});

Debug Mode

Enable debug logging in browser console:

ESSAddons.setDebug(true);

Check selected add-ons:

ESSAddons.getSelectedAddons();

Force refresh add-ons:

ESSAddons.refresh();

Change product ID dynamically:

ESSAddons.setProductId('12345678');

Testing Checklist

  • Select a framed variant → Box Frame upgrade appears
  • Select an unframed variant → Paper Upgrade appears
  • Check/uncheck add-ons → Price updates in cart
  • Click Add to Cart → Both main product and add-ons added
  • Test from a restricted country → Add-ons hidden appropriately

API Reference

The widget calls this endpoint:

GET /api/addons?productId={id}&variant={title}&size={size}&country={code}

Parameters:

  • productId - Shopify product ID (for product-specific restrictions)
  • variant - Selected variant title (e.g., "A3 Black Frame")
  • size - Print size extracted from variant
  • country - 2-letter country code (e.g., "GB", "US")
